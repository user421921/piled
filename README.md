# Pi-LED
Automation script for controlling the Pi 0's (Pi 0, 0w, 02w) Power LED

### to install:

> git clone https://github.com/user421921/piled.git

> cd piled

> cp piled.sh /bin/piled # (or where ever you wanna put it)

> chmod +x /bin/piled

> piled (on or off)
